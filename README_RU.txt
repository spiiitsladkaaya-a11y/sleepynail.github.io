Sleepy Nails - готовый статичный сайт

Что внутри:
- index.html - главная страница сайта
- styles.css - стили сайта
- assets/ - изображения

Что изменено:
- убраны админ-панель и контакты
- онлайн-запись ведет в VK: https://vk.com/club239961457
- подключены шрифты Rubik Spray Paint, Rubik Bubbles и Comic Relief
- декоративный шрифт Rubik Spray Paint используется только для крупного логотипа
- Rubik Bubbles используется для заголовков и кнопки записи
- Comic Relief используется для основного текста

Как открыть:
1. Распакуйте ZIP.
2. Откройте файл index.html в браузере.

Как разместить сайт:
Можно загрузить папку sleepy_nails_site на любой хостинг, Netlify, Vercel или GitHub Pages.

Важно:
Шрифты подключены через Google Fonts, поэтому для их загрузки нужен интернет.
